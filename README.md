## Project Overview

Metabolic Reset Clinic offers personalized metabolic health and wellness programs.  
The original website was text-heavy, visually outdated, and lacked clear conversion paths.

This redesign modernizes the brand presence while improving usability, accessibility, and trust through better layout, messaging, and visual hierarchy.

---

## Project Goals

- Improve brand credibility and professionalism
- Simplify navigation and user flow
- Clearly communicate services and benefits
- Increase appointment bookings and inquiries
- Create a scalable, modern design foundation

---

## Key Improvements

- **Modern UI/UX:** Clean layout, improved spacing, and readable typography  
- **Clear Service Structure:** Services presented with benefit-driven copy  
- **Stronger CTAs:** Strategic calls to action for consultations and bookings  
- **Trust Signals:** Testimonials, credentials, and wellness positioning  
- **Responsive Design:** Optimized for desktop, tablet, and mobile  

---

## Design Direction

The redesign follows a **clean, calming wellness aesthetic**:
- Nature-inspired color palette
- Minimal, professional typography
- Soft visual hierarchy to reduce cognitive load
- Conversion-focused layout patterns
